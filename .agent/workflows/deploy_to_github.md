---
description: Automates the process of staging all changes, committing with a timestamped message, and pushing to the remote repository.
---

1. Stage all changes
// turbo
```bash
git add .
```

2. Commit changes
// turbo
```bash
git commit -m "chore: auto-save via lutagu agent"
```

3. Push to remote
// turbo
```bash
git push
```
